About FHIRVINE SMART Proxy

FHIRVINE is a modular SMART on FHIR Single Sign-On (SSO) Proxy designed to facilitate secure authentication and authorization for FHIR-based applications.

Features
  • Register and manage SMART on FHIR applications.
  • Test client and server-side OAuth2 flows.
  • Configure proxy settings and FHIR server endpoints.
  • Support for PKCE and secure token management.
